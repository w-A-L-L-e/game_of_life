# Conway's Game of life

Simple and quick implementation of Game of Life using SDL2 running on my mac.

https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life

# Running

```
make
./game_of_life
```

Press f for full screeen. Q to quit.
